# MC² Static Website - Exact Copy

This is an exact static copy of the MC² rally team website (https://mc2.by/)

├── assets/
│   ├── css/

## Technologies Used

- HTML5
- Bootstrap 4.6.2
- jQuery 3.6.0
- Font Awesome (via fontawesome.css)
- Owl Carousel (for gallery)
- Lightbox (for image viewing)
- Montserrat Google Font
- Custom CSS (mc2.css)
- Custom JavaScript (custom.js, tabs.js, tabs2.js, video.js)

## Features

- Video background hero section
- Interactive feature cards with hover effects
- Tabbed content sections (jQuery UI tabs)
- Owl Carousel photo gallery
- Video presentation section
- Contact form with Google Maps
- Responsive navigation menu
- Smooth scrolling sections

## Setup

1. All files are already in place
2. Open `index.html` in a web browser
3. The site should work exactly like the original

## Notes

- All CSS and JavaScript files are downloaded from the original site
- All images are downloaded from the original site
- Media files (photos/videos) are from your local `mc2_static` directory
- The site uses the exact same structure, classes, and IDs as the original
